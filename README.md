# Management & Force System 

A Java-based system designed to manage students, courses, training companies, and job applications.

---

##  Features

* Manage students and users
* Handle courses and training companies
* Apply for jobs and track applications
* Organized object-oriented design

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP) / Netbeans.
